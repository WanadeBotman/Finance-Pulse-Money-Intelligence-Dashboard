Finance Pulse is a sophisticated, client-side personal finance management tool designed to provide deep insights into spending habits, budget health, and savings goals. Built with modern web technologies, it offers a responsive, "app-like" experience for tracking financial well-being directly in the browser.

## ‚ú® Features

* **Comprehensive Tracking**: Log income and expenses with specific categories, amounts, and custom notes.
* **Budget Management**: Set monthly spending limits per category with visual progress bars that change color (Warning/Exceeded) based on your real-time data.
* **Savings Goals**: Create and track progress toward specific financial targets like an Emergency Fund or a new car.
* **Visual Analytics**: 
    * **Spending Breakdown**: Interactive doughnut charts for category distribution.
    * **Income vs. Expenses**: Side-by-side bar comparisons.
    * **6-Month Trends**: Line graphs visualizing your financial trajectory over time.
* **Intelligent Insights**: Receive dynamic feedback and "Health Scores" based on your budget adherence and spending patterns.
* **Data Portability**: Export your transaction history to **CSV** or generate a professional **PDF report** directly from the dashboard.
* **Dark Mode**: Full support for a high-contrast, eye-friendly dark mode interface.
* **Mobile Optimized**: Fully responsive design with specific optimizations for iOS and Android, including "no-zoom" focus on inputs.

## üì¶ Installation & Usage

Since Finance Pulse is a static web application, no server-side setup or `npm install` is required.

1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/your-username/finance-pulse.git](https://github.com/your-username/finance-pulse.git)
    ```
2.  **Open the project**:
    Simply open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).

## üìä Data Privacy

All data is stored locally in your browser's `localStorage`. Your financial information remains private on your device and is never transmitted to, or stored on, any external servers. This makes it a secure, offline-first tool for your personal finances.
